# アップデート開発ロードマップ (2025-12-24)

## 1. 開発の全体像
本ドキュメントは、2025年12月24日に定義された機能追加およびアップデートの実装計画を管理する。

## 2. 実装スケジュール

### Phase 1: AI基盤 & DB拡張 (最優先)
AIの「記憶」と「人格」を実現するための基盤を構築する。
- [ ] データベーススキーマの拡張 (`user`テーブルへの項目追加、`user_profiles`テーブルの新設)
- [ ] AIペルソナ（ミナ/サユリ/アリス）のプロンプトエンジニアリング
- [ ] チャット履歴の永続化（過去ログの表示機能）
- [ ] ユーザー初期設定画面（目標・理由・ペルソナ選択）の作成

### Phase 2: ゲーミフィケーション (モチベーション維持)
レベルシステムとモテ度メーターを実装し、継続の価値を可視化する。
- [ ] XP（経験値）計算ロジックの実装
- [ ] 称号システムの実装
- [ ] 進化型モテ度メーター（0-30レベル）のUI構築
- [ ] レベルアップ・ステージアップ時の演出（アニメーション）

### Phase 3: 日記分析 & 習慣化ツール (機能拡充)
日々の振り返りと、誘惑への対抗手段を強化する。
- [ ] 日記のAI分析・要約APIの実装
- [ ] 瞑想・深呼吸タイマーコンポーネントの作成
- [ ] 筋トレカウンター & 習慣チェックリストの実装

### Phase 4: PWA & ブラッシュアップ
- [ ] PWA対応 (manifest.json, icon, service worker)
- [ ] 全体的なUI/UXの微調整

## 3. 技術的負債・考慮事項
- AIのメモリ消費（トークン制限）への対策。
- データの整合性（ストリークとXPの同期）。
- モバイルファーストな操作感の徹底。
